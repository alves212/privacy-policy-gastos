Privacy Policy

Last updated: July 2025

This app, GASTOS, is a personal finance tracker developed and maintained by David Alves da Silva for personal and private use only.

Information Collection and Use

The app collects only the data entered by the user, such as income, expenses, and login credentials (email and password). This data is securely stored in Firebase Firestore and is only accessible to the authenticated user.

Data Storage

All user data is stored in Firebase and protected by security rules that ensure only the account owner can access their own data.

Data Sharing

No user data is shared with third parties. As the developer, I have technical access to Firebase infrastructure, but I do not intentionally access any user's data.

Security

The app uses email and password authentication, along with Firebase's built-in security features to protect user information.

Data Deletion

Users can delete their account at any time. This will permanently remove all associated data from Firebase.

Changes to This Policy

This privacy policy may be updated from time to time. Significant changes will be communicated within the app.
